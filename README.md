
# Wardrobe App || Svelte app that uses Firebase and Openweather API 
| App that gives the user a way to organize their wardrobe by upload pictures of their clothes, check the weather and make  lists on clothes they need/changes. Uses Openweather API and FirebaseÂ´s storage, database and Authentication.

## Quick start 

*Note that you will need to have [Node.js](https://nodejs.org) installed.*

Install the dependencies...

```bash
npm install
```

...then start [Rollup](https://rollupjs.org):

```bash
npm run dev
```

## Other downloads used 

Install them...

```bash
npm i svelte-typewriter

npm i axios
```



## API
